Word Guess is a game I made using only vanilla Javascript and Bootstrap. The user has to guess which word is being randomly picked from an array of words taken from David Bowie's 'Space Oddity'.
The user enters the guesses by typing on the keyboard. You can only guess incorrectly 10 times, otherwise you lose the game! There are counters that let you play again while keeping track of how many times you've won, as well as an option to reset the game entirely.

Frameworks Used

  • Bootstrap

Languages Used

  • HTML
  • CSS
  • JavaScript

Steve Job's Trivia is a web based trivia game that tests your knowledge on Steve Jobs trivia! After reading his biography, I was shocked to find out things such as his appreciation for LSD, his relationship with Wozniak, and that he was adopted at a young age! Because of this, I decided to make a trivia game about him, and used JQuery to inject html elements to the DOM so the user doesn't have to constantly load new web pages.

Frameworks Used

  • JQuery
  • Bootstrap

Languages Used

  • HTML
  • CSS
  • JavaScript
